## IndiaBuild.tech

HackerNews like Link Aggrigation / Sharing.

Live: https://indiabuild.tech

### Development

#### Stack

- Solid.js
- SolidStart
- TailwindCSS

After cloning

1. Install the dependencies

```bash
pnpm install
```

2. Run Dev server

```bash
pnpm dev
```

Start Listenting on http://127.0.0.1:5555

### Docker setup

Required

- Docker
- Docker Compose

```bash
docker-compose up
```
